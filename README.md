# Calendar App

## Inicialización del Proyecto
1. Clonar el repositorio
2. Instalar las dependencias ('npm i/npm install')
3. Iniciar la aplicación (npm run dev)

## Dependencias
- React
- TypeScript
- Tailwind CSS
- Zustand
- Zod
- date-fns
- axios
